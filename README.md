## ▶️ Run from Source (All Platforms)

    Requirements: Python 3.10–3.12, pip.

### From repo root
    python -m pip install --upgrade pip
    pip install -r requirements.txt


    Run the game:

### Windows/macOS/Linux (from repo root)
    python src/UserInterface.py
